CORE FEATURE — INTERACTIVE CALCULATOR (MOST IMPORTANT)

• Include charts (loan balance over time, net worth over time, PSLF vs standard comparison)
• Be visually polished, fast, and intuitive

• Interest rate
• Loan type (federal/private)
• Residency length
• Specialty (dropdown with salary presets)
• Starting salary
• Salary growth %

Outputs must include:
• Monthly payment
• Total interest paid
• Time to payoff
